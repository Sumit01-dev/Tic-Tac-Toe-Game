# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. Play with a friend and enjoy the classic game in your browser!

## 📌 Features

- 🧠 2-player mode (human vs. human)
- 🎨 Responsive and clean UI
- 🔁 Restart game button
- ✅ Win/draw detection with highlight
- ⚡ Smooth interaction and instant results

---


## 🛠️ Built With

- **HTML** – for structure
- **CSS** – for styling and layout
- **JavaScript** – for game logic and interactivity
